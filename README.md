# HRReport · 人员数据看板

帮助 HRBP 在 5 分钟内清楚团队人员现状，识别人员风险的数据看板。

**演示地址：** https://j9727227-lgtm.github.io/hr-report

## 背景

HR 团队长期依赖手动拉表，招聘、绩效、离职数据分散在多个系统，无法及时且有效地了解团队现状。适用于 HRBP、HR 高层和团队 Leader。

## 解决方案

提供五张数据卡片（在岗人数、离职率、入职、预警人数等），支持按团队/职级/司龄下钻，AI 自动生成诊断和行动建议；用户可从异常指标一路追溯到员工详情页，并留下带时间戳的共享备注。
